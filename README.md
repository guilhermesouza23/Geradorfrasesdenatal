# Gerador de Frases de Natal

Este é um projeto simples que gera frases de Natal ao clicar em um botão.

## Como Usar

- Abra o arquivo `index.html` em um navegador.
- Clique no botão "Gerar Frase" para ver uma frase de Natal.
